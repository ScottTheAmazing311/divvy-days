# Divvy Days - Quick Start Guide

## Play the Game

Simply open `index.html` in your web browser. No installation required!

```bash
# Option 1: Open directly
open index.html

# Option 2: Use Python's built-in server (recommended for best experience)
python3 -m http.server 8000
# Then visit: http://localhost:8000
```

## Relationship Tips

- Each character needs **40+ points** to reach "Attempting" status
- Positive choices typically give +5 to +10 points
- Negative choices can cost -5 to -10 points
- You need 5 characters at "Attempting" or better for top endings
- Focus on: Andre, Jordan, Woody, Ramona, and Colby (easiest to befriend)

## Save System

- **Auto-save**: Game saves after every choice
- **Manual save**: Use Save button for 3 additional save slots
- **Continue**: On page reload, you can continue from auto-save
- **New Game+**: Endings unlock across playthroughs (check Endings Gallery)

## Mobile Play

- Save often in case you close the browser tab

## Troubleshooting

**Game won't load?**
- Make sure JavaScript is enabled
- Try a different browser (Chrome recommended)
- Check browser console for errors (F12)

**Progress not saving?**
- Check if localStorage is enabled
- Some browsers block localStorage in private/incognito mode
- Try regular browsing mode

**Ending seems wrong?**
- Check your final stats in the ending screen
- Review relationship scores (need 40+ for "Attempting")
- Core experiences require specific choices, not just presence
